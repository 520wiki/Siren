# Siren
WordPress Theme

Siren 是基于 Akina 主题修改而来的，感谢[Fuzzz](http://www.akina.pw/)写出这么棒的主题（[查看原始主题](http://www.akina.pw/themeakina)）。
应该属于Akina的娱乐版本吧，我也不知道为什么要改成这样，可能因为好看吧。
